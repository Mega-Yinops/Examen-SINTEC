🔍 Buscador de Pokémon

Busca tu Pokémon favorito por nombre o ID

🚀 Características Implementadas - Buscador Pokémon

🔍 Funcionalidades Principales

  • Input de búsqueda con botón "Buscar"
  • Búsqueda por nombre o ID de Pokémon
  • Consumo de PokéAPI con fetch y async/await
  • Validación de búsquedas vacías
  • Mensajes de error amigables
  • Indicador de carga "Cargando..."

🎴 Cards Interactivas

  • Cards con animación flip (CSS transform)
  • Cara frontal: imagen + nombre + tipos
  • Cara trasera: estadísticas base completas
  • Clic para rotar entre caras
  • Animaciones suaves y fluidas
  • Hints visuales para interacción

📊 Información Mostrada

  • Nombre y número del Pokémon
  • Imagen frontal (sprite oficial)
  • Tipo(s) con badges coloridos
  • Peso y altura formateados
  • Estadísticas: HP, Attack, Defense, Speed
  • Barras visuales proporcionales

⚡ Optimizaciones Técnicas

  • Caché en localStorage (sin repetir llamadas)
  • Caché inteligente por nombre, ID y variantes
  • JavaScript puro (sin frameworks)
  • Programación orientada a objetos
  • Manejo de errores robusto
  • Fallback para imágenes no disponibles

🎨 Diseño y UX

  • Diseño responsive (móvil y desktop)
  • Gradientes y efectos visuales modernos
  • Animaciones de entrada y transiciones
  • Colores temáticos Pokémon
  • Tipografía clara y legible
  • Estados visuales para feedback

🛠️ Tecnologías Utilizadas

  • HTML5 semántico y estructurado
  • CSS3 con Flexbox y Grid
  • JavaScript ES6+ (clases, async/await)
  • API REST (PokéAPI v2)
  • LocalStorage para persistencia
  • Media queries para responsive design
